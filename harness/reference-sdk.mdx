The Jami Harness developer interface is a **TypeScript SDK**, a [CLI](/harness/reference-cli),
and versioned [contracts](/harness/packages) — **not a REST API**. This page will document the
SDK surface: creating a harness, defining and gating tools, and driving runs, approvals,
resumption, and inspection.

## Planned reference

- `createHarness()` and the configuration it accepts.
- Tool gateway construction and typed [tool](/harness/concept-tools) registration.
- Starting, approving, resuming, and inspecting [runs](/harness/concept-runs).

```bash
npm install @jami-studio/harness-sdk
```
