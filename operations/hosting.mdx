# Hosting And Docs Routing

Jami Registry uses one public hostname with two backing services:

- `registry.jami.studio/registry.json`, `/items/*`, `/suites/*`, `/harness/*`, and the generated workbench/workspace preview routes are static files served by Cloudflare Pages.
- `registry.jami.studio/docs/*` is the user-facing docs route and should proxy to Mintlify.

Marketing stays at `jami.studio`.

## Current Status

- Static registry hosting is live at `https://registry.jami.studio/registry.json`.
- The npm packages are published at `0.1.0`.
- `https://registry.jami.studio/docs` is served by the Pages advanced-mode worker in `public/_worker.js`.
- `https://registry.jami.studio/` serves the generated Studio UI workbench preview.
- `https://registry.jami.studio/suites/solo/`, `/suites/business-ops/`, `/suites/mixed-media/`, and `/suites/research-writing/` serve generated workspace previews.
- `https://registry.jami.studio/harness/` serves the static Harness workbench plus status/control JSON routes generated from the Harness source repo.

## Cloudflare Subpath Proxy

Route `registry.jami.studio/docs*` through the Pages advanced-mode Worker in `public/_worker.js`.
The worker keeps Mintlify as the docs origin for general docs and normalizes legacy docs index
routes to canonical pages:

- `/docs/registry` and `/docs/registry/` redirect to `/docs/registry/overview`.
- `/docs/workbench` and `/docs/workbench/` redirect to `/docs/operations/hosting`, which
  records the hosted workbench route and unsupported persistence/backend claims.
- `/docs/suites` and `/docs/suites/` redirect to `/docs/studio/workspaces`.

Use the real Mintlify hostname from the project settings. The current origin is `jami.mintlify.dev`:

```js
const DOCS_ORIGIN = "jami.mintlify.dev";
const PUBLIC_HOST = "registry.jami.studio";

export default {
  async fetch(request, env) {
    const url = new URL(request.url);

    if (url.pathname === "/docs" || url.pathname.startsWith("/docs/")) {
      const redirectPath = docsRedirectPath(url.pathname);
      if (redirectPath) {
        const redirectUrl = new URL(request.url);
        redirectUrl.pathname = redirectPath;
        redirectUrl.search = "";
        return Response.redirect(redirectUrl.toString(), 308);
      }

      const upstream = new URL(url);
      upstream.hostname = DOCS_ORIGIN;

      const proxyRequest = new Request(upstream, request);
      proxyRequest.headers.set("Host", DOCS_ORIGIN);
      proxyRequest.headers.set("X-Forwarded-Host", PUBLIC_HOST);
      proxyRequest.headers.set("X-Forwarded-Proto", "https");

      return fetch(proxyRequest);
    }

    return env.ASSETS.fetch(request);
  },
};

function docsRedirectPath(pathname) {
  const normalizedPath = pathname.replace(/\/+$/, "");
  if (normalizedPath === "/docs/registry") return "/docs/registry/overview";
  if (normalizedPath === "/docs/workbench") return "/docs/operations/hosting";
  if (normalizedPath === "/docs/suites") return "/docs/studio/workspaces";
  return null;
}
```
