Jami Studio items install with the standard `shadcn` CLI using a direct registry URL. This
page will document the install commands, the interactive selection levels (workspace down to
individual item), and how files land in your project.

```bash
npx shadcn add https://registry.jami.studio/items/button.registry-item.json
```
