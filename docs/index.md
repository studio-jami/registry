# Jami Registry

Jami Registry is the public distribution surface for Jami Studio registry items and package documentation.

Use this site for installable UI registry items, package readiness, release evidence, and integration docs. Marketing content remains on `jami.studio`.

## Public Surfaces

- Registry bundle: `https://registry.jami.studio/registry.json`
- Docs: `https://registry.jami.studio/docs` after the Mintlify subpath proxy is deployed
- Studio UI source: `studio-jami/studio-ui`
- Harness source: `studio-jami/jami-harness`
