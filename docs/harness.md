# Jami Harness

Jami Harness owns runtime, SDK, contracts, policy, tool gateway, local provider, memory, artifacts, local store, observability, CLI, and release-readiness evidence.

Harness package readiness currently includes local package dry-runs, clean local tarball install smokes, generated SBOM evidence, and GitHub CI attestations for generated release artifacts.

Public npm publish and hosted harness runtime surfaces require trusted publishing and hosted target configuration before they can be claimed.
